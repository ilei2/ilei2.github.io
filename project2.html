Music Visualization

The music visualizer generates animation based on the frequencies of the music.The imagery is rendered in real time and synchronizes with the music as it is played.

Functions include play, pause, and volume adjustments.

Doggr

Doggr is a group project completed in CS 411 (Database Systems). The application matches users with various dog breeds based on selected preferences. Users also have the option look up shelter dogs based in Illinois.

Technology used: MySQL, PHP, Python, Wikipedia dataset, rescueme.org, petwave.com

Code and more details can be found at github
